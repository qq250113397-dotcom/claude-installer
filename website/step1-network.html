电脑端网络准备

这一页只放电脑端最关键的内容，适合先把基础网络跑通，后面的账号和工具步骤会更顺。

必做

连通外网（电脑端）

完成以下 4 步，电脑即可访问所有国际站点。

1

下载代理软件

根据你的设备选择对应版本，电脑端先完成下载再继续。

💻 电脑端（必须先完成）
🪟 Windows
🪟 Windows 64位（推荐）
✅ 支持 Windows 10 / 11（64位），如果系统较旧可去版本列表里手动选择。
🍎 macOS
不确定芯片？Apple 图标 → 关于本机，看是 M 系列还是 Intel。
🍎 macOS ARM64 🍎 macOS Intel
苹果手机小火箭说明：Shadowrocket 需要在美区 App Store 购买，注册美区 ID 的细节放在后面的手机视频里。
2

先选订阅方案，再导入外网

安装好 v2rayN 后，先看你要走付费的极速云，还是免费的 CF 方案，再按下面对应的教程导入。

💎 亲测好用付费机场

适合想省时间的人，直接用现成订阅，不用自己折腾节点来源。

  • 点 极速云 去开通
  • 开通后，按平台给你的订阅地址导入 v2rayN

🆓 CF 免费方案（备用免费方案）

如果你想先零成本试试，就走下面这套免费教程，订阅链接和导入步骤都给你留好了。

  • 适合先试用、预算紧的人
  • 保留当前免费的 CF 订阅链接
免费方案视频：下面这个节点导入演示，适合第一次导入订阅的人先照着看一遍。
免费方案说明：看完视频后，再复制下方订阅链接导入。

如果你只需要静态住宅IP / 稳定梯子，可以直接点击下面两个链接，亲测可用。

🪟 Windows / 🍎 macOS 导入步骤点击展开 ▼
  1. 打开 v2rayN
  2. 订阅分组 → 订阅设置 → 添加
  3. 粘贴订阅链接，点确定
  4. 更新全部订阅后选择低延迟节点
3

开启系统代理 / Tun 模式

首次配置：打开系统代理和 Tun 模式后再继续。
  1. 自动系统代理
  2. Tun 模式
  3. Core 类型 = sing_box
4

验证网络连通

在浏览器访问下面两个站点，能打开说明基础网络已就绪。记得用美国节点设为活动。

打开 Cloudflare ↗ 打开 dnshe ↗ 打开 ping0.cc ↗
