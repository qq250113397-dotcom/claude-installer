连通外网（电脑端）

完成以下 4 步，电脑即可访问所有国际站点。

1

下载代理软件

根据你的设备选择对应版本，电脑端先完成下载再继续。

💻 电脑端（必须先完成）
🪟 Windows
🪟 Windows 64位（推荐）
✅ 支持 Windows 10 / 11（64位），如果系统较旧可去版本列表里手动选择。
🍎 macOS
不确定芯片？Apple 图标 → 关于本机，看是 M 系列还是 Intel。
🍎 macOS ARM64 🍎 macOS Intel
2

导入订阅链接，连通外网

安装好 v2rayN 后，将下方订阅链接导入即可。

看这个演示就够了：下面的视频就是“节点导入”，适合第一次导入订阅的人先照着看一遍。
https://zuibenniuge.cc.cd/sub?token=b05b99657be99d182b612ab1c1caf641
🪟 Windows / 🍎 macOS 导入步骤点击展开 ▼
  1. 打开 v2rayN
  2. 订阅分组 → 订阅设置 → 添加
  3. 粘贴订阅链接，点确定
  4. 更新全部订阅后选择低延迟节点
3

开启系统代理 / Tun 模式

首次配置：打开系统代理和 Tun 模式后再继续。
  1. 自动系统代理
  2. Tun 模式
  3. Core 类型 = sing_box
4

验证网络连通

在浏览器访问下面两个站点，能打开说明基础网络已就绪。
