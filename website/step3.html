步骤三：登录 & 开始使用

在终端启动 Claude Code，按提示完成浏览器授权，即可开始使用。

🔑 在终端中登录

Claude Code 安装成功后，首次运行会引导你完成登录授权。

1

打开终端

🪟 Windows：按 Win + R，输入 powershell，按回车。

🍎 macOS：打开 Terminal（终端）。

2

运行 Claude Code

输入以下命令启动：

claude

首次运行会弹出登录提示。

3

按提示完成浏览器授权

终端会显示登录链接，用浏览器打开并完成授权：

Claude Code v1.x.x

? 请在浏览器中完成登录 →
  https://claude.ai/login?code=XXXXXX

等待授权完成...

✓ 登录成功！欢迎使用 Claude Code。
4

开始使用

登录成功后，在任意项目目录打开终端，输入 claude 即可开始使用。
