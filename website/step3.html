步骤三：登录 & 开始使用

在终端完成登录授权后即可开始使用。下面把登录、分流规则和价格说明放在同一页，减少来回跳转。

🔑 在终端中登录

首次运行 claude 会弹出浏览器登录授权，按提示完成即可。

1

打开终端

🪟 Windows：按 Win + R，输入 powershell，按回车。

🍎 macOS：按 ⌘ + 空格，输入「终端」打开。

2

运行 Claude Code

输入以下命令启动：

claude

首次运行会弹出登录提示，按浏览器里的授权流程完成即可。

3

按提示完成浏览器授权

终端会显示登录链接，用浏览器打开并完成授权：

Claude Code v1.x.x

? 请在浏览器中完成登录 →
  https://claude.ai/login?code=XXXXXX

等待授权完成...

✓ 登录成功！欢迎使用 Claude Code。
4

开始使用

登录成功后，在任意项目目录打开终端，输入 claude 即可开始使用。

🎉

恭喜，流程完成！

你已完成全部流程，可以开始使用 Claude Code / Codex 了。

遇到问题？查看 FAQ 常见问题 获取帮助。
