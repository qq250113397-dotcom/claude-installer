步骤三：注册 & 开始使用

前两步完成后，这一步负责把账号注册好，并把 Claude / GPT / Codex 的登录流程收口。

👤 注册账号

如果你已经有 Gmail，可以直接跳过这一步，进入后面的登录流程；没有账号的话，先按视频完成手机端注册。

1

访问注册页面

确保外网已经连通，先完成你要使用的账号注册；Claude / GPT / Codex 都可以按各自官网提示继续，注册演示见配套视频。

2

选择注册方式

推荐使用 Google 账号登录，一键授权更方便。点击「Continue with Google」，选择你注册好的 Gmail 账号即可完成授权。

为什么推荐 Google 账号？ 一键授权，无需设置密码，登录更快捷，且与 Claude Code 终端登录流程完全兼容。
3

完成验证

按提示完成邮箱验证或账号授权，同意服务条款后即注册成功。

🔑 在 Claude Code 中登录

账号注册完成后，在终端中完成 Claude Code 登录；GPT / Codex 也可以按同样思路使用你自己的账号继续安装。

1

打开终端

🪟 Windows 用户：按 Win + R，输入 powershell，按回车。

🍎 macOS 用户：打开 Terminal（终端），输入 claude 即可。

2

运行 Claude Code

输入以下命令启动 Claude Code：

claude

首次运行会弹出登录提示。

3

按提示完成浏览器授权

终端会显示如下提示，按照指引在浏览器中完成授权：

Claude Code v1.x.x

? 请在浏览器中完成登录 →
https://claude.ai/login?code=XXXXXX

等待授权完成...

✓ 登录成功！欢迎使用 Claude Code。
4

开始使用

登录成功后，你可以在任意项目目录打开终端，输入 claude 开始使用。

🎉
