步骤三：登录 & 开始使用

注册或登录 Anthropic 账号，将 Claude Code 与你的账号关联，即可开始使用。

👤 注册 Anthropic 账号

如果你已有账号，可跳过此部分直接登录。

1

访问注册页面

确保代理已开启，在浏览器中打开 claude.ai，点击「Sign Up」按钮。

2

选择注册方式

推荐使用 Google 账号登录，一键授权更方便，无需记密码。点击「Continue with Google」，选择你注册好的 Gmail 账号即可完成授权。

为什么推荐 Google 账号？ 一键授权，无需设置密码，登录更快捷，且与 Claude Code 终端登录流程完全兼容。
3

完成验证

按提示完成邮箱验证或账号授权，同意服务条款后即注册成功。

🔑 在 Claude Code 中登录

账号注册完成后，在终端中完成登录授权：

1

打开终端

🪟 Windows 用户：按 Win + R，输入 powershell，按回车。

🍎 macOS 用户：打开 Terminal（终端），同样输入 claude 即可。

2

运行 Claude Code

输入以下命令启动 Claude Code：

claude

首次运行会弹出登录提示。

3

按提示完成浏览器授权

终端会显示如下提示，按照指引在浏览器中完成授权：

Claude Code v1.x.x

? 请在浏览器中完成登录 →
https://claude.ai/login?code=XXXXXX

等待授权完成...

✓ 登录成功！欢迎使用 Claude Code。
📸 截图：终端显示登录链接，浏览器打开授权页面
4

开始使用

登录成功后，你可以在任意项目目录打开终端，输入 claude 开始使用。

API Key 方式： 高级用户也可使用 API Key 登录。在 console.anthropic.com 创建 API Key，然后运行 claude config set apiKey YOUR_KEY 进行配置。

🚀 快速使用指南

登录成功后，试试这些常用命令：

claude
启动交互式聊天

直接对话，询问编程问题或让 Claude 帮你写代码

claude "任务描述"
单次任务执行

快速让 Claude 完成一个具体任务，不进入交互模式

claude --help
查看帮助文档

显示所有可用命令和选项的完整说明

claude --version
查看当前版本

显示已安装的 Claude Code 版本号

🎉

恭喜，安装完成！

你已成功安装并配置好 Claude Code，现在可以开始 AI 辅助编程之旅了。
