步骤一：网络准备

Claude Code 需要访问境外服务器，在安装前请先确保网络可以正常连接。

为什么需要这一步？
目标是先把外网连通，后续无论是注册 Claude 账号、安装 Claude Code，还是搭建自己的订阅，都需要稳定的境外网络。 下载 v2rayN 并导入下方订阅链接，即可一键完成配置。
必做

连通外网（电脑端）

完成以下 4 步，电脑即可访问所有境外站点。

1

下载代理软件

根据你的设备选择对应版本，均从 GitHub 官方下载最新版：

🪟 Windows 64位 🍎 macOS ARM64（M 系列） 🍎 macOS Intel（老款） 📱 安卓（NekoBox）
🛠️ 常见安装问题 点击展开 ▼

🪟 Windows 常见问题

1. 提示"Windows 已保护你的电脑" → 点击 "更多信息" → 再点 "仍要运行"
2. 提示缺少运行库 → 安装 .NET 6.0 Runtime：dotnet.microsoft.com/download/dotnet/6.0
3. 杀毒软件误报 → 先解压到普通文件夹，再将 v2rayN 加入白名单
4. 解压后找不到 exe → 右键 ZIP 选 "解压到当前文件夹"，再双击文件夹里的程序

🍎 macOS 常见问题

1. 无法打开 → 在 Finder 里右键 v2rayN.app，点 "打开"；弹窗后再点一次 "打开"
2. 还是被拦截 → 到 系统设置 → 隐私与安全性，找到 "仍要打开" 并确认
3. 以上无效时，才用终端：xattr -dr com.apple.quarantine <路径>

官方版不含预置节点，下载后参考步骤 2 导入订阅链接即可获得可用节点。

导入订阅链接，连通外网

安装好 v2rayN 后，将下方订阅链接导入，即可获得可用节点：

🪟 Windows v2rayN — 导入订阅步骤 点击展开 ▼
  1. 解压下载的 ZIP，双击文件夹内的 v2rayN.exe 启动
  2. 点顶部菜单 「订阅分组」→「订阅设置」
  3. 点 「添加」，将上方订阅链接粘贴进地址栏，点 「确定」
  4. 回主界面，点 「订阅分组」→「更新全部订阅」，等待节点列表刷新
  5. 点顶部闪电图标 ⚡（测速），等延迟数字出现
  6. 双击延迟最低的节点，点工具栏 「系统代理」→「自动配置系统代理」
🍎 macOS v2rayN — 导入订阅步骤 点击展开 ▼
  1. 打开 DMG，把 v2rayN.app 拖入 应用程序（Applications）
  2. 从应用程序打开 v2rayN（参考上方安装问题跳过 Gatekeeper 拦截）
  3. 点菜单栏图标 → 「订阅分组」→「订阅设置」
  4. 点 「添加」，粘贴上方订阅链接，点 「确定」
  5. 「订阅分组」→「更新全部订阅」，等节点列表刷新
  6. 全部测速后双击延迟低的节点，点 「系统代理」→「自动配置系统代理」
3

开启系统代理 / Tun 模式

首次配置：官方版需要手动开启以下三个设置，添加好节点后再点连接。

请确认这三个开关都已打开：

  1. 自动系统代理
  2. Tun 模式
  3. Core 类型 = sing_box
4

验证网络连通

系统代理开启后，在浏览器访问以下两个站点，能打开说明基础网络已就绪：

打开 Cloudflare ↗ 打开 dnshe ↗

2

住宅 IP 就绪后，注册 Claude 账号

链式代理接入住宅 IP 后，浏览器走的是真实的美国家庭宽带 IP，注册 Claude 不会被拦截：

  1. 确认住宅 IP 已连通，访问 ping0.cc 确认 IP 显示为美国
  2. 在浏览器打开 claude.ai
  3. 点击「Continue with Google」，选择你的 Gmail 账号完成注册
✓ 完成标志：能正常访问 claude.ai，账号注册成功，即可进入步骤二安装 Claude Code。

🔧 常见连接问题

🐢 连接很慢怎么办？

  • 在 v2rayN 里全部测速，切换到延迟最低的节点
  • 点「订阅分组 → 更新全部订阅」刷新节点列表
  • 关闭其他占用网络的程序

❌ 连接超时怎么办？

  • 检查代理工具是否已启动并连接节点
  • 确认系统代理或 Tun 模式已开启
  • 重启代理工具后重新导入订阅
