步骤一：网络准备

Claude Code 需要访问境外服务器，在安装前请先确保网络可以正常连接。

为什么需要这一步？
这一步的目标只有一个：先把网络连通到你后续要用的两个站点，Cloudflare 和 dnshe。 我们这里优先提供一个预配置便携版，让你先少走几步。

📋 网络配置检查清单

按照以下步骤完成配置，每完成一步请在脑海中打勾：

1

下载代理软件

根据你的设备选择对应的代理工具下载安装，Windows 和 macOS 都提供了预置节点、自动系统代理、Tun 模式和分流规则的便携版：

📱 安卓手机（NekoBox） 🍎 macOS 便携版 🪟 Windows 便携版
🚀 苹果手机（Shadowrocket）
🛠️ 常见安装问题 点击展开 ▼

🪟 Windows 常见问题

1. 提示"Windows 已保护你的电脑" → 点击 "更多信息" → 再点 "仍要运行"
2. 提示缺少运行库 → 需要安装 .NET 6.0 Runtime，下载地址：dotnet.microsoft.com/download/dotnet/6.0
3. 杀毒软件误报拦截 → 先把安装包重新解压到普通文件夹，再将 V2RayN 加入白名单；不要直接在压缩包里运行
4. 解压后找不到 exe → 右键压缩包选择 "解压到当前文件夹"，再双击文件夹里的程序启动

🍎 macOS 常见问题

1. 不想用终端 → 在 Finder 里右键 v2rayN.app，点 "打开"；第一次弹窗再点一次 "打开"
2. 还是被拦截 → 到 系统设置 → 隐私与安全性，在下面找到 "仍要打开" 并确认
3. 只有在上面两种都不行时，再用终端命令：xattr -dr com.apple.quarantine ... 和 chmod +x ...
🍎 苹果手机用户请注意：
Shadowrocket（小火箭）需在 美区 AppStore 购买下载，无法直接提供安装包。
注册美区 Apple ID 需要在电脑网页端切换至美国节点后完成，苹果手机需要单独操作，具体步骤见步骤 3 的视频教程。

下载后按视频教程安装配置，工具需要支持系统代理模式（让电脑所有软件都走代理网络）。

Windows 和 macOS 便携版都已预置启动器、初始节点、自动系统代理、Tun 模式和分流规则，并且在打包时已经自动测过延迟，预选了一个可用的 US 节点。解压后直接双击启动即可；这一步不需要手动导入旧订阅。

2

先连通外网

Windows 和 macOS 便携版已经预置了初始节点，打开后直接点一次连接即可。

3

注册谷歌邮箱（Gmail）

💡 已有谷歌账号？直接跳到第 4 步，无需重复注册。

外网连通后，注册一个 Gmail 邮箱。后续无论是自己搭建订阅、买住宅 IP，还是注册 Claude / GPT / Codex，都会用到它。

⚠️ 请用手机 App 注册，不要用电脑网页：
在中国大陆用网页注册 Gmail 会要求短信验证，国内手机号无法通过验证，会注册失败。
需要先翻墙，再在手机上打开 Gmail App，通过 App 完成注册。

5

电脑开启系统代理 / Tun 模式

已预设：Windows 和 macOS 便携版已经默认打开 自动系统代理、Tun 模式，Core 类型也已经改好 sing_box。包里还会先自动测延迟，默认预选一个可用的 US 节点。你打开软件后，通常只要点一次连接就行。

如果你只想先跑起来，不用管下面这些高级选项；便携版已经帮你设好了。

请确认这几个开关都已经打开

  1. 自动系统代理
  2. Tun 模式
  3. Core 类型 = sing_box

下方截图保留给你对照位置，打开软件后先确认这几个开关已经打开，再点连接即可。

6

用谷歌账号注册 Claude 账号，进入步骤二

网络验证通过后，前往 Claude 官网，用谷歌账号完成注册：

  1. 1 确保代理已开启，打开浏览器访问 claude.ai
  2. 2 点击「Continue with Google」，选择刚注册的 Gmail 账号
  3. 3 填写用户名，完成注册，即可开始免费使用
✓ 完成标志： 能够在浏览器中正常访问 claude.ai， 说明网络已配置完成，可以进行下一步。

🔧 常见问题

🐢 连接很慢怎么办？

  • 尝试切换节点（代理服务器的位置）
  • 选择延迟（响应时间，越低越好）低的境外节点
  • 关闭其他占用网络的程序

❌ 连接超时怎么办？

  • 检查代理工具是否已启动运行
  • 确认已开启系统代理（全局模式）
  • 重启代理工具后重试步骤 4
