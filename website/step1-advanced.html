搭建住宅 IP 链式代理

如果你要继续推进后续账号流程，这一页就是重点。

1

用 Gmail 登录 Cloudflare，搭建订阅

视频涵盖以下操作：
  • 用 Gmail 登录 Cloudflare 和 dnshe
  • 部署 CF Workers 并绑定自己的域名
  • 将自己的订阅导入 v2rayN / NekoBox / Shadowrocket
  • 验证可用后，继续下一步
📎 视频中所需资源一览
Cloudflarehttps://dash.cloudflare.com/login

住宅 IPOnesProxy
IP 检测https://ping0.cc
Codex 快速开始https://developers.openai.com/codex/quickstart
Codex 邀请规则https://developers.openai.com/codex/pricing
2

住宅 IP 就绪后，注册 Claude 账号

接入住宅 IP 后，浏览器走的是真实线路，后续注册会顺很多。

  1. 确认住宅 IP 已连通，访问 ping0.cc
  2. 打开 claude.ai
  3. 点击 Continue with Google，选择 Gmail 账号完成注册
✓ 完成标志：能正常访问 claude.ai，账号注册成功，即可进入步骤二安装 Claude Code / Codex 相关工具。若后面也要用 Codex，先把上方官方链接收藏起来。
