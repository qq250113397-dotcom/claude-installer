什么是 Claude Code？

Claude Code 是由 Anthropic 开发的官方命令行 AI 编程助手，可以直接在你的终端中运行。 它能理解你的代码库、编写和修改代码、运行命令、调试错误，是目前最强大的 AI 编程工具之一。

💡 核心功能

  • 智能代码补全与生成
  • 自动调试和错误修复
  • 代码解释与重构建议
  • 多文件项目理解

🖥️ 系统要求

  • Windows 10 / 11 (64位)
  • 网络连接（需访问境外）
  • 至少 500MB 磁盘空间
  • Anthropic 账号（免费注册）
