什么是 Claude Code？

Claude Code 是 Anthropic 开发的官方命令行 AI 编程助手，直接在终端中运行。 能理解你的代码库、编写和修改代码、运行命令、调试错误，是目前最强大的 AI 编程工具之一。

💡 核心功能

  • 智能代码生成与补全
  • 自动调试和错误修复
  • 代码解释与重构建议
  • 多文件项目理解

🖥️ 系统要求

  • Windows 10/11 或 macOS 12+
  • 稳定的网络连接（需访问国际站点）
  • 至少 500MB 磁盘空间
  • Node.js 18 LTS 或更高
