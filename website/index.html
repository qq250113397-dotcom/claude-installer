这套教程的网络方案

本站教程默认使用 CF Workers 节点 + 静态住宅 IP 的链式代理方案。 这样做的好处是，注册和登录 Claude 时通常更稳定，不容易跳额外验证；如果你主要只用 GPT / Gemini，只搭好 CF Workers 机场节点就够了， 不必额外购买住宅 IP，也可以直接开始使用。

如果你是新手，直接跟着步骤走就行，页面会带你把该点的都点完；如果你已经会自己搭链式代理， 可以直接去 步骤一第 4 点 点击进入 OnesProxy 购买静态住宅 IP，自行接到你自己的环境里。

什么是 Claude Code？

Claude Code 是由 Anthropic 开发的官方命令行 AI 编程助手，可以直接在你的终端中运行。 它能理解你的代码库、编写和修改代码、运行命令、调试错误，是目前最强大的 AI 编程工具之一。

💡 核心功能

  • 智能代码补全与生成
  • 自动调试和错误修复
  • 代码解释与重构建议
  • 多文件项目理解

🖥️ 系统要求

  • Windows / macOS 全平台
  • 网络连接（需访问境外）
  • 至少 500MB 磁盘空间
  • Anthropic 账号（免费注册）
