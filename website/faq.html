常见问题 FAQ

汇总了安装和使用过程中的常见问题，点击问题查看答案。

🌐 网络相关

为什么必须要有境外网络才能用 Claude Code？ ▾
第一阶段先连通境外网络，是为了能打开 Cloudflare 和 dnshe，方便后面搭建自己的订阅和链式代理。 后续再继续注册和安装 Claude / GPT / Codex。
安装程序运行时提示「网络连接失败」怎么办？ ▾
请按以下步骤排查：

1. 确认步骤一里的便携版已经启动，并且初始节点处于连接状态
2. 先尝试打开 Cloudflare 和 dnshe
3. 如果页面打不开，再回到代理工具里检查「系统代理」或「Tun 模式」是否已开启
4. 重新运行安装程序
安装成功后每次使用都需要开代理吗？ ▾
只要你还要继续访问 Cloudflare、dnshe、Claude / GPT / Codex 这些境外站点，就需要保持代理工具处于开启状态。 建议将代理工具设置为开机自启，这样后续不用每次手动打开。

📦 安装相关

安装时 Windows 提示「无法运行此应用」或 SmartScreen 警告怎么办？ ▾
这是因为安装程序尚未通过微软的代码签名认证，属于正常现象。解决方法：

1. 点击弹窗中的「更多信息」
2. 点击「仍要运行」按钮

本程序代码完全开源，你可以在 GitHub 上查看和审计所有代码，确认安全性。
安装后运行 claude 提示「不是内部或外部命令」怎么办？ ▾
这是 PATH 环境变量未更新的问题。解决方法：

1. 重启终端： 完全关闭 PowerShell/CMD 窗口，重新打开
2. 重启电脑： 如果重启终端无效，重启电脑后再试
3. 手动验证： 运行 npm list -g @anthropic-ai/claude-code，如果有输出说明安装成功，只是 PATH 问题
电脑上已有 Node.js，但版本太旧怎么办？ ▾
Claude Code 需要 Node.js 18 或更高版本。安装程序会自动检测版本，如果版本过低会提示你更新。 你也可以手动从 nodejs.org 下载最新 LTS 版本安装后，再运行 Claude Code 安装程序。
如何卸载 Claude Code？ ▾
在 PowerShell 中运行以下命令即可完全卸载：

npm uninstall -g @anthropic-ai/claude-code

Node.js 本身不会被删除，如需一并卸载，去「控制面板 → 程序和功能」中找到 Node.js 卸载即可。

🔑 登录 & 账号

Claude Code 是免费的吗？ ▾
Claude Code 软件本身免费，但使用需要 Anthropic 账号，并消耗 API 用量。

计费方式： 按 Token 用量计费，新用户通常有一定免费额度。实际费用取决于使用量， 轻度使用每月大约几美元，重度使用可能更多。

详细定价请参考 Anthropic 官方定价页面。
登录时浏览器不会自动打开怎么办？ ▾
如果浏览器未自动打开，终端会显示一个登录链接。手动复制该链接到浏览器地址栏打开， 完成授权后终端会自动检测并完成登录。
注册账号需要手机号或信用卡吗？ ▾
注册账号不需要手机号和信用卡，使用 Google 或 Apple 账号即可完成注册。 如果你想使用付费 API 功能，才需要绑定支付方式（支持信用卡）。 新用户通常有免费试用额度，可以先体验再决定是否充值。

🔄 更新相关

如何更新到最新版本的 Claude Code？ ▾
运行安装目录中的 update.bat 脚本即可自动更新。或者手动在终端运行：

npm update -g @anthropic-ai/claude-code

建议定期更新以获得最新功能和安全修复。
安装程序本身如何获取更新？ ▾
安装程序通过 Git 进行版本管理。运行 update.bat 会自动从 GitHub 拉取最新版本的安装脚本， 然后重新安装最新版 Claude Code。你也可以关注发布页面获取更新通知。
没有找到你的问题？ 欢迎在 GitHub Issues 中提问，或参考 Claude Code 官方文档 获取更详细的说明。
