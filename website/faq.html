🌐 网络相关

为什么必须要联网才能用 Claude Code？ ▾
第一阶段先连通外网，是为了能打开 Cloudflare 和 dnshe，方便后面搭建自己的订阅和链式代理。 后续再继续注册和安装 Claude / GPT / Codex。
安装程序运行时提示「网络连接失败」怎么办？ ▾
请按以下步骤排查：

1. 确认 v2rayN 已启动，且已导入订阅链接并选中了一个低延迟节点
2. 检查「系统代理」或「Tun 模式」是否已开启
3. 先尝试打开 Cloudflare 和 dnshe，能打开说明网络已通
4. 重新打开终端，再次运行 npm 安装命令
安装成功后每次使用都需要开代理吗？ ▾
只要你还要继续访问 Claude / GPT / Codex 这些国际站点，就需要保持代理工具处于开启状态。 建议将代理工具设置为开机自启，这样后续不用每次手动打开。
CF 节点老是断线 / 频繁掉线怎么办？ ▾
CF Workers 免费节点本质上是借用 Cloudflare 的公共 IP，会受到运营商 QoS 限速和 CF 自身的并发限制，断线属于正常现象。几个缓解方法：

1. 断了就切换节点：在 v2rayN 里点顶部闪电图标全部测速，双击当前延迟最低的节点切换即可，不用重启软件。

2. 定期更新订阅：「订阅分组 → 更新全部订阅」刷新节点列表，新节点通常比旧的稳定。

3. 换成住宅 IP（根本解决）：CF 节点适合临时上网，长期稳定使用建议在步骤一进阶部分接入住宅 IP（OnesProxy），住宅 IP 走真实家庭宽带线路，稳定性远高于 CF 免费节点。

📦 安装相关

安装时 Windows 提示「无法运行此应用」或 SmartScreen 警告怎么办？ ▾
这是因为安装程序尚未通过微软的代码签名认证，属于正常现象。解决方法：

1. 点击弹窗中的「更多信息」
2. 点击「仍要运行」按钮
安装后运行 claude 提示「不是内部或外部命令」怎么办？ ▾
这是 PATH 环境变量未更新的问题。解决方法：

1. 重启终端： 完全关闭 PowerShell/CMD 窗口，重新打开
2. 重启电脑： 如果重启终端无效，重启电脑后再试
3. 手动验证： 运行 npm list -g @anthropic-ai/claude-code，如果有输出说明安装成功，只是 PATH 问题
电脑上已有 Node.js，但版本太旧怎么办？ ▾
Claude Code 需要 Node.js 18 或更高版本。从 nodejs.org 下载最新 LTS 版本安装后，再运行安装命令即可。
如何卸载 Claude Code？ ▾
在终端运行以下命令即可完全卸载：

npm uninstall -g @anthropic-ai/claude-code

Node.js 本身不会被删除，如需一并卸载，去「控制面板 → 程序和功能」中找到 Node.js 卸载即可。

🔑 登录 & 账号

Claude Code 是免费的吗？ ▾
Claude Code 软件本身免费，但正常使用需要订阅 Claude Pro（$20/月） 或 Claude Max（$100/月），没有免费使用额度。

国内用户可通过 WildAI 等平台完成充值订阅，具体操作见步骤三。

详细定价请参考 Anthropic 官方定价页面。
为什么 CF 免费节点无法注册 Claude 账号？ ▾
CF Workers 免费节点使用的是 Cloudflare 的公共 CDN IP，这些 IP 被 Claude / Google 识别并拦截，因此无法完成注册或登录。

解决方法：完成步骤一进阶部分，接入住宅 IP（OnesProxy），住宅 IP 走真实家庭宽带线路，可以正常注册和使用 Claude。
登录时浏览器不会自动打开怎么办？ ▾
如果浏览器未自动打开，终端会显示一个登录链接。手动复制该链接到浏览器地址栏打开， 完成授权后终端会自动检测并完成登录。
注册账号需要手机号或信用卡吗？ ▾
注册账号不需要手机号和信用卡，使用 Google 账号即可完成注册。 订阅 Claude Pro 需要信用卡或通过 WildAI 等第三方平台充值。

🔄 更新相关

如何更新到最新版本的 Claude Code？ ▾
在终端运行以下命令即可更新到最新版本：

npm update -g @anthropic-ai/claude-code

建议定期更新以获得最新功能和安全修复。
Node.js 如何更新？ ▾
前往 nodejs.org 重新下载 LTS 版本安装即可，新版本会覆盖旧版本。
