步骤二：下载 & 安装

下载一键安装程序，双击后自动完成 Node.js、Claude Code 和必要依赖的安装。

🖥️

Claude Code 一键安装程序

适用于 Windows 10 / 11（64 位），双击后自动安装所有依赖，下载失败会自动切换国内镜像

⬇️ 下载安装程序 (.exe)
📦 约 2MB 🪟 Windows 10/11 ✅ 无病毒，开源可审计 🔄 支持自动更新
🍎

Claude Code macOS 安装助手

适用于 macOS，双击后自动安装所有依赖，网络不稳时会自动重试并切换镜像

⬇️ 下载安装脚本 (.sh)
🍎 Apple Silicon / Intel ✅ 无病毒，开源可审计
安全提示： 下载完成后，Windows 可能弹出 SmartScreen 警告，点击「更多信息」→「仍要运行」即可。 本程序开源，源代码可在 GitHub 查看审计。
🍎 macOS 用户：下载后双击 .sh 文件，如提示无法打开，前往系统设置 → 隐私与安全性 → 点击「仍要打开」

📦 安装步骤

运行安装程序后，按以下流程操作：

1

双击运行安装程序

Windows 直接双击下载好的 install.bat；macOS 直接双击解压后的 install-mac.sh，按提示允许执行即可。

2

自动检查并补齐依赖

安装器会自动检查 Node.js、npm 和网络环境；缺什么就自动下载并补齐，不需要你手动配置 PATH。

3

自动安装 Claude Code

程序会先走国内镜像，失败后再回退官方源，安装完成后会自动给出下一步提示。

4

安装完成后直接用

看到安装成功提示后，打开新的终端输入 claude 即可开始使用。

5

遇到问题怎么办？

如果安装失败，先确认步骤一里的网络已经连通，再回到 FAQ 查对应报错。大多数问题都是网络或权限导致的。

💻 系统要求

⚙️ 硬件要求

  • Windows 10 版本 1903 或更高
  • 64 位操作系统
  • 至少 500MB 可用磁盘空间
  • 稳定的网络连接

📦 将自动安装

  • Node.js 22 LTS（若未安装）
  • npm 包管理器
  • @anthropic-ai/claude-code
  • PATH 环境变量配置
安装失败？ 最常见原因是网络问题。请确认：①代理工具已开启 ②系统代理模式已启用 ③重新运行安装程序。 更多问题请查看 FAQ 页面。
