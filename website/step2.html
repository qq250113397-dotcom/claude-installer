步骤二：下载 & 安装

先完成 Node.js 安装，再用一行命令完成 Claude Code 基础安装。Codex 的官方入口也单独放出来，方便同时保留两条路线。

Node.js for Windows

下载 LTS 版本，双击 .msi 安装包按向导完成安装，全程点「Next」即可

⬇️ 前往 Node.js 官网下载
🪟 Windows 10/11 ✅ 官方来源 选 LTS 版本

打开 Claude 桌面版

📦 安装 Node.js 后，打开终端执行以下命令完成 Claude Code 基础安装：

npm install -g @anthropic-ai/claude-code

Windows 请用 命令提示符（cmd） 或 PowerShell；macOS 请用终端（Terminal）。

❓ 不知道怎么打开终端？点这里 点击展开 ▼

🪟 Windows — 打开命令提示符 / PowerShell

方法一（推荐）：按键盘 Win + R，在弹出框里输入 cmd，按回车
方法二：点击开始菜单，搜索「PowerShell」，点击打开
方法三：在文件夹空白处按住 Shift + 右键，选「在此处打开 PowerShell 窗口」

🍎 macOS — 打开终端（Terminal）

方法一（推荐）：按 ⌘ Command + 空格 打开 Spotlight，输入「终端」或「Terminal」，按回车
方法二：打开「访达（Finder）」→「应用程序」→「实用工具」→「终端」
打开后：看到黑色或白色窗口并有光标闪烁，就说明终端已就绪。把上方命令复制粘贴进去，按回车即可。
提示： Node.js 安装完成后，打开新的终端窗口再运行 npm 命令，确保环境变量已刷新。
🪟 Windows 常见安装问题 点击展开 ▼

❌ 问题一：npm 报网络错误 / 下载超时

v2rayN 开启后 npm 仍可能无法联网，因为 npm 不自动读取系统代理。
v2rayN 默认 HTTP 代理端口是 10809（可在 v2rayN → 参数设置 → Core 设置里确认）。

在 PowerShell / cmd 中依次运行：

npm config set proxy http://127.0.0.1:10809
npm config set https-proxy http://127.0.0.1:10809
npm install -g @anthropic-ai/claude-code

安装完成后如需恢复：npm config delete proxy 和 npm config delete https-proxy

❌ 问题二：PowerShell 提示"无法运行脚本"

Windows 默认禁止运行外部脚本，需要先放开执行策略：

Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

输入后选 Y 确认，再重新运行 npm 安装命令。

❌ 问题三：安装完后 claude 命令找不到

完全关闭当前终端窗口，重新打开一个新的 PowerShell / cmd，再输入 claude 即可。若还是找不到，重启电脑后再试。

🍎 macOS 常见安装问题 点击展开 ▼

❌ 问题一：npm 报权限错误（EACCES / permission denied）

macOS 默认全局安装需要管理员权限。在命令前加 sudo 即可，输入电脑密码后按回车（密码不会显示出来，是正常的）：

sudo npm install -g @anthropic-ai/claude-code

❌ 问题二：npm 报网络错误 / 下载超时

macOS 上 npm 也不会自动读系统代理。常见代理软件默认 HTTP 端口：
• v2rayN（Mac 版）：10809　• ClashX / Surge / Stash：7890

在终端依次运行（端口换成你自己的）：

npm config set proxy http://127.0.0.1:7890
npm config set https-proxy http://127.0.0.1:7890
sudo npm install -g @anthropic-ai/claude-code

安装完成后恢复：npm config delete proxy 和 npm config delete https-proxy

❌ 问题三：安装完后 claude: command not found

先关掉当前终端，重新打开一个再试。还是不行的话，把对应路径加到 PATH：

Apple Silicon（M1/M2/M3/M4）芯片：

echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc

Intel 芯片：

echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc

不确定芯片？点左上角 Apple 图标 → 关于本机查看。

❌ 问题四：双击 .pkg 装不上，提示「无法打开」

系统偏好设置 → 隐私与安全性 → 滚到底部，看到「已阻止 Node.js」点「仍要打开」即可。

📦 安装步骤

按以下流程完成安装：

1

下载并安装 Node.js LTS

点击上方「前往 Node.js 官网下载」，选择 LTS（长期支持） 版本下载。Windows 双击 .msi，macOS 双击 .pkg，按向导一路 Next / 继续 即可。

2

打开终端 / 命令提示符

Windows：按 Win + R，输入 cmd 回车；或搜索「PowerShell」打开。
macOS：按 ⌘ + 空格 搜索「终端」打开。

3

运行安装命令

在终端中粘贴以下命令并回车，等待安装完成：

npm install -g @anthropic-ai/claude-code
4

验证安装成功

安装完成后，打开新的终端窗口，输入 claude --version，能看到版本号说明安装成功，随后输入 claude 即可开始使用。

5

遇到问题怎么办？

如果 npm 命令报错，先确认步骤一里的网络已经连通，再回到 FAQ 查对应报错。常见原因是网络未连通或 Node.js 安装后未重新打开终端。

💻 系统要求

⚙️ 系统要求

  • Windows 10 / 11（64 位）或 macOS 12+
  • Node.js 18 LTS 或更高
  • 至少 500MB 可用磁盘空间
  • 稳定的网络连接（已完成步骤一）

📦 需要安装

  • Node.js LTS（从 nodejs.org 下载）
  • npm（随 Node.js 一起安装）
  • @anthropic-ai/claude-code（npm 命令）
安装失败？ 最常见原因是网络问题。请确认：① 代理工具已开启 ② 系统代理模式已启用 ③ 关闭终端重新打开后再运行 npm 命令。 更多问题请查看 FAQ 页面。
