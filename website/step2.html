步骤二：下载 & 安装

前往 Node.js 官网下载并安装，随后通过一行 npm 命令完成 Claude Code 安装。

🖥️

Node.js 官方安装包（Windows）

前往 Node.js 官网下载 LTS 版本，双击安装包按向导完成安装，全程点「Next」即可

⬇️ 前往 Node.js 官网下载
🪟 Windows 10/11 ✅ 官方来源 📦 选 LTS 版本
🍎

Node.js 官方安装包（macOS）

前往 Node.js 官网下载 LTS 版本的 .pkg 安装包，双击安装即可

⬇️ 前往 Node.js 官网下载
🍎 Apple Silicon / Intel ✅ 官方来源 📦 选 LTS 版本

📦 安装 Node.js 后，打开终端执行以下命令安装 Claude Code：

npm install -g @anthropic-ai/claude-code

Windows 用户请用 命令提示符（cmd） 或 PowerShell；macOS 用户请用终端（Terminal）。

提示： Node.js 安装完成后，打开新的终端窗口再运行 npm 命令，确保环境变量已刷新。

📦 安装步骤

按以下流程完成安装：

1

下载并安装 Node.js LTS

点击上方「前往 Node.js 官网下载」，选择 LTS（长期支持） 版本下载。Windows 双击 .msi 安装包，macOS 双击 .pkg 安装包，按向导一路 Next / 继续 即可。

2

打开终端 / 命令提示符

Windows：按 Win + R，输入 cmd 回车；或搜索「PowerShell」打开。
macOS：按 ⌘ + 空格 搜索「终端」打开。

3

运行安装命令

在终端中粘贴以下命令并回车，等待安装完成：

npm install -g @anthropic-ai/claude-code
4

验证安装成功

安装完成后，打开新的终端窗口，输入 claude --version，能看到版本号说明安装成功，随后输入 claude 即可开始使用。

5

遇到问题怎么办？

如果 npm 命令报错，先确认步骤一里的网络已经连通，再回到 FAQ 查对应报错。常见原因是网络未连通或 Node.js 安装后未重新打开终端。

💻 系统要求

⚙️ 系统要求

  • Windows 10 / 11（64 位）或 macOS 12+
  • Node.js 18 LTS 或更高
  • 至少 500MB 可用磁盘空间
  • 稳定的网络连接（已完成步骤一）

📦 需要手动安装

  • Node.js LTS（从 nodejs.org 下载）
  • npm 随 Node.js 一起安装
  • @anthropic-ai/claude-code（npm 命令）
安装失败？ 最常见原因是网络问题。请确认：①代理工具已开启 ②系统代理模式已启用 ③关闭终端重新打开后再运行 npm 命令。 更多问题请查看 FAQ 页面。
