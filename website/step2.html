📦 安装步骤

运行安装程序后，按以下流程操作：

1

以管理员身份运行

右键点击下载好的 install.bat，选择「以管理员身份运行」，允许 UAC 弹窗。

2

自动检测环境

安装程序会自动检测：

  • Node.js 是否已安装（版本需 ≥ 18）
  • 网络代理配置（自动检测常用端口）
  • 磁盘剩余空间是否充足
3

自动安装 Node.js（如需要）

如果未检测到 Node.js，程序将自动下载并安装 Node.js LTS 版本（约 30MB），完成后自动继续。

4

安装 Claude Code

通过 npm 全局安装 Claude Code：

npm install -g @anthropic-ai/claude-code

此步骤需要访问境外 npm 源，请确保网络代理已就绪（约 1-3 分钟）。

5

安装完成

看到「✅ 安装成功！」提示后，Claude Code 已安装完毕。打开新的 PowerShell 窗口，输入 claude 即可启动。

6

遇到问题怎么办？

安装过程中遇到任何报错，可打开 Claude 桌面版，截图发给 Claude 描述问题，Claude 会根据你的具体报错给出解决方案，比搜索引擎更快更准确。

💻 系统要求

⚙️ 硬件要求

  • Windows 10 版本 1903 或更高
  • 64 位操作系统
  • 至少 500MB 可用磁盘空间
  • 稳定的网络连接

📦 将自动安装

  • Node.js 22 LTS（若未安装）
  • npm 包管理器
  • @anthropic-ai/claude-code
  • PATH 环境变量配置
安装失败？ 最常见原因是网络问题。请确认：①代理工具已开启 ②系统代理模式已启用 ③重新运行安装程序。 更多问题请查看 FAQ 页面。
